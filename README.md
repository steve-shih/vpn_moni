# VPN 差異檢查器 (VPN Diff Checker)

這是一個 Google Chrome 擴充功能 (Manifest V3)，旨在幫助使用者快速檢查目前的網路環境與瀏覽器指紋資訊，並與先前儲存的「基準值 (Baseline)」進行比對，從而判定 VPN 是否有效以及改變了哪些可見資訊。

## 核心功能

- **環境偵測**：自動取得公網 IP、地理位置 (國家/地區/城市)、ISP 供應商。
- **指紋收集**：收集時區、語言、User-Agent、平台、螢幕解析度及 WebGL 渲染器資訊。
- **差異比對**：將目前資訊與儲存的基準值比對，標示「已改變」、「未改變」或「未知」。
- **智能摘要**：根據比對結果自動產生判定摘要（例如：僅改變網路出口、變更了指紋等）。
- **基準設定**：一鍵將當前環境設為基準值，方便後續比對。

## 安裝教學

1. 下載本專案原始碼。
2. 開啟 Google Chrome 瀏覽器，進入「擴充功能」頁面 (`chrome://extensions/`)。
3. 開啟右上角的「開發者模式」。
4. 點擊「載入解壓縮擴充功能」。
5. 選擇本專案的根目錄 (`vpn-diff-checker` 資料夾)。

## 使用流程

1. **建立基準**：
   - 在尚未開啟 VPN 的情況下，開啟擴充功能。
   - 點擊「設為目前基準」按鈕。
   - 系統會將目前的 IP 與指紋資訊存入 `chrome.storage.local`。

2. **進行檢查**：
   - 開啟 VPN 或變更網路環境。
   - 點擊擴充功能圖示。
   - 系統會自動比對並在上方顯示「目前判定」。
   - 下方列表會詳列每個欄位的變動狀態。

## 檔案結構

- `manifest.json`: 擴充功能定義檔 (MV3)。
- `popup.html`: 彈出視窗介面。
- `popup.css`: 介面樣式。
- `popup.js`: 主要邏輯控制。
- `utils/`: 模組化工具函式。
    - `ipService.js`: IP 與 Geo 抓取。
    - `fingerprintService.js`: 瀏覽器資訊收集。
    - `compareService.js`: 比對規則與摘要生成。
    - `storageService.js`: 儲存管理。

## 技術說明

- **Manifest V3**：遵循最新的 Chrome 擴充功能規範。
- **隱私聲明**：所有比對邏輯皆在本地執行，且資料僅儲存在使用者的瀏覽器本地空間 (`chrome.storage.local`)，不會上傳至任何遠端伺服器。
- **IP API**：使用 `api.ipify.org` 與 `ipinfo.io` 作為 IP 來源。
